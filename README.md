# git-branch-selector

![screenshot](./media/screenshot.png)

## Install

```bash
pip install git-branch-selector
```
or
```bash
python3 -m pip install git-branch-selector
```

## Usage

```bash
git-branch-selector
```

- arrow-up / down / j / k to move cursor
- enter to select branch
- q to quit
- ctrl-c to quit
